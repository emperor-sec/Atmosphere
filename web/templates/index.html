Atmosphere

An ethical OSINT reconnaissance panel for IP geolocation and device fingerprinting.

Overview

Atmosphere is a lightweight, self-hosted OSINT panel written in Go. It provides self-scan diagnostics, manual and batch IP geolocation lookup with automatic provider fallback, and DNS/reverse DNS resolution. It is designed for security awareness demonstrations, infrastructure reconnaissance training, and personal network diagnostics.

Capabilities
Self ScanPublic IP, local IP discovery, browser, OS, device type, client hints
IP LookupManual geolocation lookup with Google Maps and OpenStreetMap links
Batch LookupUp to 20 IP addresses resolved in a single request
DNS ToolsForward hostname resolution and reverse PTR lookup
Geolocation FallbackAutomatically tries multiple providers for higher accuracy
Ethical Use

Atmosphere does not collect data from third parties without their knowledge. Every lookup in this panel operates only on the visitor's own connection or on an IP, hostname, or address the operator explicitly enters. There is no tracking-link or covert-capture mechanism. Use this tool only on systems you are authorized to test.
